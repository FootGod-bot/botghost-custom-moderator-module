# /ban Command.
## REQUIREMENTS
To change the config of this command, the /config command is needed
## Step 1: Create the case number variable
Create a new coustom command  
Click the second tab on the right.  
Create a new variable  
Name it mod_case_number  
Set variable type to Number  
Set default value to 0  
Add the volume  
## Step 2: Create the user cases variable
Create a new variable  
Name it mod_user_cases  
Set variable type to User
Set user type to Collection
Change nothing else  
Add the volume  
## Step 3: Create the warn count variable
Create a new variable  
Name it mod_warn_count  
Set variable type to User  
Set user type to Number  
Set the default to 0  
Add the volume  
## Step 4: Create the mute count variable
Create a new variable  
Name it mod_mute_count  
Set variable type to User  
Set user type to Number  
Set the default to 0  
Add the volume  
## Step 5: Create the action variable
Create a new variable  
Name it mod_action  
Set variable type to Object  
Leave everything else as is  
Add the volume  
## Step 6: Create the moderator variable
Create a new variable  
Name it mod_moderator  
Set variable type to Object  
Leave everything else as is  
Add the volume  
## Step 7: Create the reason variable
Create a new variable

Leave everything else as is  
Add the volume  
## Step 8: Create the target variable
Create a new variable

Set variable type to Object  
Leave everything else as is  
Add the volume  
## Step 9: Create the time variable
Create a new variable  
Name it mod_time  
Set variable type to Object

Add the volume  
## Step 10: Import the command
Go to the first tab  
Click import and insert the code for the version you need under version. The current latest is CMD_f33ee13e-e85a-4ff3-98bc-9dbe9a754bdb  
All done
