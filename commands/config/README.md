# /config Command.
## REQUIREMENTS
No other commands are required for this command to work
## Step 1: Create the config variable
Create a new coustom command \n
Click the second tab on the right. \n
Create a new variable \n
Name it mod_config \n
Set variable type to Object \n
Change nothing else \n
Add the volume \n
## Step 2: Import the command
Go to the first tab \n
Click import and insert the code \n
